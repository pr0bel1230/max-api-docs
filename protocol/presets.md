# Пресеты: аватары, стикеры, эмодзи

MAX предоставляет API для получения предустановленных аватаров,
стикеров и эмодзи.

## GET_PRESET_AVATARS (opcode 25)

Получение списка предустановленных аватаров для профиля.

### Запрос

```json
{}
```

Payload может быть пустым объектом. Поля `count` и `offset` игнорируются —
всегда возвращается полный список.

### Ответ

```json
{
  "presetAvatars": [
    {
      "name": "Антро",
      "avatars": [
        {
          "url": "https://i.oneme.ru/i?r=...",
          "id": 3778498
        },
        {
          "url": "https://i.oneme.ru/i?r=...",
          "id": 3722368
        }
      ]
    }
  ]
}
```

| Поле | Тип | Описание |
|------|-----|----------|
| `presetAvatars` | array | Массив категорий аватаров |
| `presetAvatars[].name` | string | Название категории (например, "Антро") |
| `presetAvatars[].avatars` | array | Массив аватаров в категории |
| `avatars[].url` | string | URL изображения аватара |
| `avatars[].id` | int | ID аватара |

### Особенности

- Изображения хранятся на `i.oneme.ru`
- Категории включают различные темы: "Антро" (антропоморфные),
  животные, абстракции и т.д.
- ID аватара можно использовать для установки аватара профиля

## GET_PRESETS (opcode 26)

Получение предустановленных наборов: стикеры, эмодзи, реакции.

### Запрос

```json
{
  "type": "sticker",
  "marker": 0,
  "count": 10
}
```

| Поле | Тип | Описание |
|------|-----|----------|
| `type` | string | Тип пресета (см. таблицу) |
| `marker` | int | Смещение для пагинации |
| `count` | int | Количество элементов |

### Типы пресетов

| Значение | Ответ | Описание |
|----------|-------|----------|
| `sticker` | `{"stickers": [...], "marker": 0}` | Стикеры |
| `emoji` | `{"emojis": [...], "animojis": [...], "marker": 0}` | Эмодзи |
| `reaction` | `{"animojis": [...], "marker": 0}` | Реакции (анимированные) |

### Пример ответа (sticker)

```json
{
  "stickers": [],
  "marker": 0
}
```

### Пример ответа (emoji)

```json
{
  "emojis": [],
  "animojis": [],
  "marker": 0
}
```

### Особенности

- **Пагинация:** поле `marker` в ответе можно использовать для
  получения следующей страницы (передать в следующий запрос)
- **Пустые массивы:** на момент исследования возвращались пустые
  массивы — возможно, требуется дополнительное поле `packId` или
  другой payload
- **Неподдерживаемые типы:** `gif`, `card`, `file`, `image`, `video`,
  `audio`, `location`, `contact`, `reply`, `forward`, `link`, `poll` —
  возвращают ошибку `"No enum constant"`
